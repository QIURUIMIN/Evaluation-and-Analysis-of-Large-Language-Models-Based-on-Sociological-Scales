已有的大语言模型 / Existing Large Language Models

  • 讯飞星火1.0 / Spark 1.0

    星火大模型1.0是讯飞推出的初版，主要提供基础的大语言模型功能，包括文本理解、生成等基本能力，适用于通用的语言处理任务。

  • 讯飞星火2.0 / Spark 2.0

    在星火大模型2.0中，科大讯飞增加了代码能力和多模态能力。这意味着模型不仅可以处理文本，还能理解和生成代码，以及处理图像数据。例如，模型可以理解图片内容并据此生成文本或进行故事叙述，同时在代码生成和补全方面进行了优化，显著提高了效率和准确性。

  • 讯飞星火3.0 / Spark 3.0

    星火大模型3.0则进一步专注于行业应用，进行了更多领域的优化，如教育、医疗、法律等专业领域。此版本更注重于实际应用中的表现，提供了更为定制化的解决方案，以满足特定行业的需求。
